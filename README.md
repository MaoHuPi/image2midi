# image2midi

by maohupi

## arguments
* ***`-i`***: input file path(must be a jpeg file)
* ***`-o`***: output file path(will be a midi file)
* ***`-m`***: convert mode('gate' or 'edge')
* ***`-y`***, ***`-n`***: overwrite confirm(to overwrite or not)
* ***`-h`***: help(see all of the arguments)
